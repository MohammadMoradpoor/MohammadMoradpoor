## System Architecture Priorities
- Orchestration-first design for reliability under retries, failures, and long-running state transitions.
- Autonomous agent systems with explicit routing, observability, and control boundaries.
- Low-latency realtime voice loops for production-grade conversational interaction.
- Machine learning and deep learning as first-class architecture components, not isolated experiments.
