## AI Engineering Profile
I design and deliver production-grade AI systems centered on **autonomous agents**, **workflow orchestration**, and **scalable LLM architecture**.  
My work spans end-to-end platform delivery: agent runtime design, orchestration reliability, retrieval infrastructure, API architecture, and high-performance product integration.

| Capability Area | Engineering Focus |
