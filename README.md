## System Architecture Priorities
- Build for **scalability first**: agent workflows must survive retries, partial failures, and long-running state transitions.
- Engineer **autonomous agents with control planes**: explicit routing, guardrails, observability, and recovery paths.
- Optimize **real-time interaction loops**: low-latency voice, streaming context, and deterministic orchestration under load.
- Treat **machine learning as a system component**: retrieval, ranking, classification, and speech pipelines integrated into production services.
